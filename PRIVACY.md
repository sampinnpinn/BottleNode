# 瓶子画布 - 隐私政策

最后更新：2026年4月

## 1. 概述

瓶子画布（以下简称"我们"）致力于保护您的隐私。本隐私政策说明了我们如何收集、使用和保护您的信息。

## 2. 数据收集

我们仅收集以下类型的数据：

- **本地存储数据：** 使用 Chrome 扩展的 storage 权限，在您的浏览器本地存储您的设置和偏好（如主题设置、历史记录等）。这些数据仅存储在您的设备上。
- **API 调用数据：** 当您使用图片生成功能时，我们会向 BananaRouter API 发送必要的请求参数，以获取生成结果。

## 3. 数据使用

我们收集的数据仅用于：

- 提供和改进瓶子画布的核心功能
- 保存您的用户设置和偏好
- 生成和处理您的图片请求

## 4. 数据存储

您的所有设置和偏好数据都存储在您的浏览器本地，不会上传到我们的服务器。我们不会访问或收集您的本地存储数据。

## 5. 第三方服务

瓶子画布使用以下第三方服务：

- **BananaRouter API：** 用于提供图片生成功能。我们仅发送必要的请求参数，不会共享您的个人信息。

## 6. 数据共享

我们不会出售、交易或租借您的个人信息。我们仅在以下情况下可能与第三方共享数据：

- 获得您的明确同意
- 法律要求或保护我们的合法权益

## 7. 数据删除

您可以随时通过以下方式删除您的数据：

- 在 Chrome 扩展管理页面中清除瓶子画布的存储数据
- 卸载瓶子画布扩展

## 8. Cookie 使用

瓶子画布不使用 Cookie 或类似的跟踪技术。

## 9. 儿童隐私

瓶子画布不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

## 10. 政策变更

我们可能会不时更新本隐私政策。重大变更时，我们会通过扩展内通知或其他方式告知您。
